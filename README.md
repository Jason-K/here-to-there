# Terminal Finder

Open currently selected Finder (or QSpace Pro, Bloom) window in **Terminal**<sup>*</sup>

### <sup>*</sup> Supported:

- [Ghostty](https://ghostty.org/)
- [iTerm2](https://iterm2.com/)
- [kitty](https://sw.kovidgoyal.net/kitty/)
- Terminal
- [Warp](https://www.warp.dev/)
- [WezTerm](https://wezterm.org/index.html)
- [QSpace Pro](https://qspace.awehunt.com/en-us/index.html)
- [Bloom](https://www.bloomapp.dev/)

### Document apps (active file -> target)

- VS Code (stable + Insiders)
- Microsoft Word, Excel, PowerPoint
- Preview
- Adobe Acrobat / Acrobat Reader DC
